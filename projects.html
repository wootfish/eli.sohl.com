obsidian +

Obsidian is a constraint-based system for graphics programming. It lets you define groups of shapes, specify relations between them, style them in any way supported by the SVG format, and render them from that information alone.

This is a new project, and it is very much a work in progress, but the early results are very promising. You can read about the project here, and you can read its source code here.

theseus +

I've been working on this project since early 2017. The goal: to create a robust system for sharing and preserving research. This was first conceived as a response to attempts by several governments to censor research on climate change.

The system consists of two parts: The actual application described above - Theseus - and a distributed hash table (DHT) on top of which the application is built. In theory, a number of existing DHT algorithms could be used, but I'm designing a new one with the working name Theseus DHT. I believe this new design can greatly improve on today's popular DHTs' defenses against certain categories of attack, particularly Sybil attacks.

These projects' implementations are not yet complete, but the theoretical results achieved so far (the best of which I'm still in the process of writing up) are striking.

cryptopals +

These are so much fun. They're a series of cryptography challenges written by some folks who really know their stuff. Sets 1 through 7 are available here, and you can check out Set 8 here. They start out easy enough, but the later sets have some genuinely tricky problems in them.

You can check out my solutions on Github. I've worked through sets 1 through 7 so far, and I plan on taking on set 8 soon.

Back in the day I was very into Project Euler (in fact, I'm still ranked in the 99.4th percentile on there after half a decade of inactivity), and a lot of what makes Project Euler fun can also be found in Cryptopals. Most programming is hard in tedious ways; this stuff is hard in fun ways. The most challenging parts of it are also the most interesting. And it's just so damn satisfying when you finish troubleshooting and get to watch your script work for the first time.
